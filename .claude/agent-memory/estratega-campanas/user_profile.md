El usuario opera PROCORPMDigital, una agencia de marketing digital IA-first en el mercado hispanohablante. Es el Orquestador del sistema multi-agente.

**Rol en el sistema**: Director / Orquestador — coordina todos los agentes y tiene decision final sobre estrategia, presupuesto y entregables.

**Preferencias de entrega observadas**:
- Documentos completos en Markdown con estructura clara de secciones
- Metadatos de agente/tarea en cada entregable
- Idioma espanol en todos los entregables
- Nomenclatura de tareas segun convencion: `{CLIENTE}-{AÑO}-{NUM}-T{TAREA}`
- Briefings creativos separados por agente de produccion

**Contexto del proyecto principal**: PROCORP es tanto el operador del sistema como el primer cliente de prueba. La agencia se autogestiona como caso de uso real.
